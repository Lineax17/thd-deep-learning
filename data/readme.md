# Data Preparation

## Download the dataset

Download the RGB Dataset

Repo Link:
https://github.com/phelber/eurosat

Direct Download Link:
https://madm.dfki.de/files/sentinel/EuroSAT.zip

## Prepare the Data

- Extract the files
- Rename the extracted folder to 'original'

Do not touch the original folder! In case of splitting the dataset, use the given splitting script that copies the data before splitting. You may adjust the split ratios towards your own use case.
