# Deep Learning Projektarbeit

## Datensatz

Unser Datensatz ist der EuroSAT Datensatz, welcher 10 verschiedene 
Klassen von Satellitenbildern enthält. Jede Klasse repräsentiert eine 
bestimmte Landnutzung oder Landbedeckung, wie z.B. landwirtschaftliche 
Flächen, Wälder, städtische Gebiete, Wasserflächen usw. Der Datensatz 
besteht aus insgesamt 27.000 Bildern, die in 10 Klassen aufgeteilt sind, 
mit jeweils 2.700 Bildern pro Klasse.
